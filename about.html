About This Site

Website Introduction

Welcome to my Swedish Language Learning Hub!
This is a personal project I created for my own learning purposes for both Swedish and programming.

Website Features

  • User Management System for tracking learning progress
  • HomePage with featured words and flashcards
  • WordPage with Table and Features: Search, Add, Delete, Edit(currently focusing on nouns)
  • Responsive designfor access on various devices

Technology Stack

This website is built using the following technologies:

  • Frontend: HTML, CSS (Bootstrap framework)
  • Backend: Node.js, Handlebars
  • Database: SQLite3
  • Authentication: bcrypt for password encryption, express-session for session management
